[PART 4: FORMULAS]
--------------------------------------------------------------------------------

Lambda (Cardinality Penalty) Computation:
  λ = clip(10 × scale × (N / K), 50, 500)
  where:
    N = number of assets in candidate universe
    K = target portfolio size
    scale = max(|diagonal terms|) of QUBO

Gamma (Sector Penalty) Computation:
  γ = 0.1 × λ

QUBO Objective Function:
  E(x) = x^T Q x
  where Q = q_risk × Σ + linear_terms + cardinality_penalty + sector_penalty

Sharpe Ratio Calculation:
  Sharpe = (μ_p - r_f) / σ_p
  where:
    μ_p = portfolio annualized expected return
    σ_p = portfolio annualized volatility
    r_f = risk-free rate (5%)
